## Requirements

### X11

You need to install xclip

### Wayland

You need to install wl-copy

## Cool libraries used

- [egui-notify](https://github.com/ItsEthra/egui-notify)
- [egui-tracing](https://github.com/grievouz/egui_tracing)
- [egui-file](https://lib.rs/crates/egui_file)
